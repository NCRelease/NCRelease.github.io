Latest Release

The latest release is version 2.04 - released August 22, 2020. It consists of the following components:

  • Hub firmware for Gen-1 hubs - version 1.04
  • Hub firmware for Gen-2 hubs - version 2.04
  • Configuration software (both the full tool and EasyLoad) - version 1.04

You can download zipped files containing these components here:

netCleConfig 1.04 Gen-1 Hub 1.04 Gen-2 Hub 2.04

A minor update to the configuration sofware - version 1.04.1 - was created December 22, 2020. There are no functional changes, but the release packaging changed. There is no longer a lib directory. All the required modules are loaded in netCleConfig.jar and netCleEasyLoad.jar, making these stand-alone programs.

netCleConfig 1.04.1

Improvements in this release

Here is a summary of the more important changes made since last April.

  • Support for the Gen-2 hub hardware.
  • The addition of the EasyLoad utility.
  • More key options for HIDSpecial and Press/Release.
  • Option to compile Gen-2 software to add a softSerial port, for remote login experimentation.
  • Creation of a socket connection option, for remote login experimentation.
  • Force exported files to have a .txt extension.
  • Expanded the options for the Joystick Mouse Solution.
  • Automatically open previously used directory for Import and Export.
  • Added a Move button to move triggers.
  • Allow the moving of triggers during an Import.
