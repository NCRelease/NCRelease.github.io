Loading Firmware

To load or upgrade the firmware on the hub you will need the Arduino IDE program. This is a free application. The Windows version is available at https://www.arduino.cc/en/software. There is lots of infomation on using the IDE on the web.

After installing the Arduino IDE you will need to add the IRLib2 library to your installation. Download the following zip file, open it, and follow the instructions in the readme.md file.

IRLib2-master.zip

Using the Arduino IDE, open the hub '.ino' file (either HubV1.ino or HubV2.ino).

Under the "Tools" menu, go to the "Board" sub-menu and select "Arduino Leonardo".

Connect your hub to your PC's USB port. Then open the IDE's serial monitor ("Tools" > "Serial Monitor") and enter a capital 'V'. If there is already netCle firmware running on the hub it will respond with the version number preceeded by 'V' and followed by 'Z' (e.g. 'V2.04Z'). This step puts the hub into IDLE mode. In this mode the hub will not interfere with the download of new firmware.

Finally, you can click on the "Upload" button (2nd from the left) to load the new firmware.
